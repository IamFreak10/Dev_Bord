Question-1: What are the different ways to select an element in the DOM?

The DOM elements can be selected using getElementById(), getElementsByClassName(), and getElementsByTagName(). Additionally, querySelector() and querySelectorAll() allow selecting elements using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML retrieves or sets the HTML content of an element, including tags, while innerText gets or sets the visible text, excluding hidden elements. textContent returns or sets the raw text content, including hidden text, and is faster than innerText.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is added to a parent element, and it handles events for its child elements. This approach improves performance and simplifies managing events, especially when elements are added or removed dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event triggered on a child element propagates upwards through its parent elements, triggering their event listeners in the process. This allows parent elements to handle events for their child elements, even if the event was initially fired on a child.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use `document.createElement()`, then add it with `appendChild()` to a parent. To remove an element, use `removeChild()` on its parent.
